# LTS 维护说明

本项目进入 LTS 长期维护阶段。

LTS 的意思是 Long-Term Support，也就是长期支持。对这个项目来说，含义是：不再做大规模架构重构，不再频繁增加新功能，主要保证项目还能被下载、阅读、运行和学习。

## 维护范围

会优先维护：

- README 运行步骤
- 依赖版本说明
- 数据库导入说明
- 明显的启动问题
- 截图与项目介绍
- 常见问题文档
- 总入口链接与项目导航

一般不再维护：

- 大规模升级 Django 主版本
- 重写前端页面
- 替换整体技术栈
- 增加复杂权限系统
- 增加大量新业务功能

## 推荐使用方式

建议把本项目作为完整 Django 数据分析项目参考：

1. 先按 README 跑通原项目。
2. 再阅读数据处理脚本、模型、视图和模板。
3. 最后替换自己的数据或增加新的分析页面。

如果你正在准备课程设计或毕业设计，建议重点说明你自己新增或改造的部分，例如数据集、指标、图表、页面、算法或部署方式。

## 问题反馈建议

提交 issue 时建议包含：

- 操作系统
- Python 版本
- Django 版本
- MySQL 版本
- 完整报错信息
- 已执行的命令
- 是否导入 SQL 文件
- 是否修改数据库配置
